Arduino Electrónica ServoMotores Bluetooth

- Resumen -

En esta esta práctica se volvera a utilizar el brazo robotico previamente hecho, implementando un modulo Bluetooth para que se pueda mover por medio de un celular sin tener que hacerlo manualmente.

- Introducción -

Introducción: En esta práctica el objetivo es poder hacer un modulo desde la app Inventor para lograr que el brazo pueda moverse a travez de una aplicación creada por nosotros y que el modulo Bluetooth controle los servomotores
