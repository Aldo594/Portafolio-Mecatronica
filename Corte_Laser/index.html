Corte Laser

Aldo Yair
Ingenieria Mecatrónica
Fecha: Agosto 30, 2024
SolidWorks Laser

- Resumen -

En esta práctica se vio cómo diseñar moldes para cortarlos en láser, nos enseñaron cómo funciona la máquina y qué materiales puede cortar, así como darle potencia y velocidad para cortar las piezas o poner un grabado; también las tolerancias que deben tener las piezas para que puedan ensamblar de forma correcta la cual es de 0.2 mm; para esta práctica teníamos que diseñar nuestras propias piezas para hacer un ensamble en la cortadora, las piezas fueron diseñadas en 2D para guardarlas en formato dwxf, y luego ponerlas en SmartCarvelInstall para poder poner las piezas y modificar lo que queríamos (potencia y velocidad). Lo que diseñamos fue un avión. Está práctica nos sirvió para saber cómo funciona la máquina y poder hacer estructuras para otros proyectos en láser.
